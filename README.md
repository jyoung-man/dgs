# Daou-Github-Search

## 프로젝트 개요
Daou-Github-Search 앱은 SwiftUI와 Combine을 사용하여 GitHub Search API를 활용한 실무형 과제 프로젝트입니다.  
사용자가 키워드를 입력하면 GitHub의 Repository를 검색하고, Star/Unstar 기능과 OAuth 로그인을 제공합니다.

### 주요 기능
- Repository 검색 (키워드 기반)
- 무한 스크롤 지원
- Star / Unstar 연동
- GitHub OAuth 로그인
- SwiftUI 기반 UI 구현 (List, TextField 커스텀 등)

- iOS 16.0 이상
